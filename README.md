# Financial Assets Dashboard

**Financial Assets Dashboard** is a portfolio project focused on building a clean, well-architected frontend application for exploring financial asset data (cryptocurrencies).

The main goals of this project are:
- Demonstrate strong Frontend development skills
- Apply real-world architectural decisions
- Showcase thoughtful state management and UI composition
- Avoid overengineering while keeping production-level practices

## Tech Stack

- React + Vite + TypeScript  
- React Router  
- TanStack Query (server state)  
- Redux Toolkit (UI state only)  
- Axios  
- shadcn/ui  
- CoinGecko API
